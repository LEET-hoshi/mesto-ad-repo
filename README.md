# Mesto Russia 🌍

Это интерактивный веб-сервис для обмена фотографиями, созданный в рамках обучения в «Яндекс Практикуме». Проект представляет собой полноценную социальную сеть для обмена визуальным контентом с возможностью управления профилем и добавления карточек.


### 🔗 [Посмотреть демо](https://leet-hoshi.github.io/mesto-ad-repo/)

---

## 🛠 Технологический стек
* **Frontend:** JavaScript (ES6+), HTML5, CSS3 (Flexbox, Grid).
* **Сборка:** Vite.
* **Архитектура:** Объектно-ориентированный подход (ООП), модульность.
* **Инфраструктура:** CI/CD на базе GitHub Actions для автоматического деплоя на GitHub Pages.

---

## 🚀 Основной функционал
* **Управление профилем:** Редактирование имени и описания «о себе».
* **Работа с контентом:** Добавление новых карточек с изображениями и описанием.
* **Взаимодействие:** Возможность ставить «лайки» и удалять свои карточки.
* **Интерактивность:** Модальные окна с валидацией форм и обработкой пользовательского ввода.
* **Адаптивность:** Корректное отображение интерфейса на различных разрешениях экранов.

---

## 🏗 Ключевые технические решения
В процессе разработки основной упор был сделан на чистоту кода и масштабируемость:
1. **ООП:** Логика разделена на модули (классы), что упрощает поддержку проекта.
2. **Валидация:** Реализована кастомная система проверки полей ввода в формах.
3. **Автоматизация:** Настроен CI/CD пайплайн (GitHub Actions), который собирает проект через Vite и автоматически деплоит его на GitHub Pages при каждом пуше в ветку `main`.

---

## 📦 Как развернуть проект локально
1. Клонируй репозиторий:
   `git clone https://github.com/leet-hoshi/mesto-ad-repo.git`
2. Перейди в папку с проектом:
   `cd mesto-ad`
3. Установи зависимости:
   `npm install`
4. Запусти проект в режиме разработки:
   `npm run dev`
